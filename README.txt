İMZA TEKNİK MÜHENDİSLİK - WEB SİTESİ
==========================================

Mekanik tesisat ve mühendislik hizmetleri sunan İmza Teknik Mühendislik'in kurumsal web sitesi.

TEKNİK ÖZELLİKLER
-----------------
• Responsive tasarım
• Modern slider sistemi
• Proje filtreleme
• Smooth scroll
• Hamburger menü
• WOW.js animasyonları

DOSYA YAPISI
------------
index.html      - Ana sayfa
services.html   - Hizmetler sayfası
projects.html   - Projeler sayfası
main.js         - Ana JavaScript
projects.js     - Proje filtreleme
services.js     - Hizmet yönetimi
style.css       - Ana stiller
img/            - Görseller

KULLANIM
--------
index.html dosyasını tarayıcıda açın veya web sunucusu kullanın.
